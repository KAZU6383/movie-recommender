# 🎬 Movie Recommender App

映画/ドラマのレビューとAIを活用して、気分にぴったりの映画/ドラマをおすすめするアプリです。

## 🌟 特徴

- 気分に合わせて映画をレコメンド（例: 「元気になりたい」「癒されたい」）
- **自然言語処理（NLP）× Sentence-BERT** で意味ベースの類似検索
- OMDb API を使って映画ポスターも自動表示
- シンプルな Streamlit UI
